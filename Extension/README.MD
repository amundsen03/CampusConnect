# 🦉 CampusConnect AI - Chrome Extension

**Din personlige AI-assistent for INN University studenter!**

## 🎯 Funksjoner

- ✅ Spesifikke svar om PC-krav for AI-studiet
- ✅ Steg-for-steg guide for studentkort/ID-kort
- ✅ Informasjon om studiet, opptak og campusliv
- ✅ Pre-promptet AI med kunnskap om INN University
- ✅ Vakkert design med CampusConnect-logo
