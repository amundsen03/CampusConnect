# Studier ved Campus Hamar

Studietilbudet endres over tid. For oppdatert liste skal du bruke INNs offisielle studieoversikt og filtrere på campus Hamar.

## Slik finner du oppdatert studieliste (Norsk/Engelsk)
1. Åpne INNs “Our studies / Våre studier”.
2. Filtrer på **Campus: Hamar**
3. Velg nivå (bachelor/master/videreutdanning) etter behov.
4. Åpne hvert studie for detaljer (opptakskrav, språk, studieplan).

## Vanlige spørsmål chatboten kan svare på (når dere fyller mer data)
- “Hvilke bachelorstudier finnes på Hamar?”
- “Finnes det engelskspråklige programmer på Hamar?”
- “Hva er opptakskravene for [studie]?”
