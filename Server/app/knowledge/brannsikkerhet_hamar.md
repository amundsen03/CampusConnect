# Brannsikkerhet for studenter

## Ved brann (standard rutine)
- Aktiver brannalarm hvis du kan gjøre det trygt.
- Ring 110.
- Evakuer via nærmeste rømningsvei (ikke bruk heis).
- Gå til oppsamlingsplass.
- Følg anvisninger fra brann-/sikkerhetspersonell.

## Husk
Du bør vite hvor rømningsveier og slokkeutstyr er i byggene du bruker.
